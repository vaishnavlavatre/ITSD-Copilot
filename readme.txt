# ITSD Admin Copilot 

A smart AI assistant for IT Service Desk Unix administration tasks.

 Quick Start

### Method 1: Simple Start (Recommended for Beginners)

1. **Start Backend:**
   - Double-click `start_backend.bat`
   - Wait for "Server started on http://localhost:5000"

2. **Start Frontend:**
   - Double-click `start_frontend.bat` 
   - Wait for "Serving HTTP on 0.0.0.0 port 8000"

3. **Open Application:**
   - Go to http://localhost:8000 in your browser
