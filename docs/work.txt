    Clone the repository and navigate to the project directory.

    Set up Python virtual environment for the backend and CLI.

    Install dependencies for the backend and CLI.

    Run the backend using python run.py or using Docker.

    Access the web UI at http://localhost:5000.

    Use the CLI by running python cli.py ask --query "your query".

>python cli.py login
